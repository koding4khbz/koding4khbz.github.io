Foire Aux Questions (FAQ)

● Mes achats n'apparaissent pas ?

Utilisez le bouton "Restaurer les achats" dans les réglages du jeu. Si le problème persiste, contactez-nous avec votre reçu Apple.

● Comment regagner des cœurs ?

Vous gagnez automatiquement 1 cœur toutes les 5 minutes (maximum 5 cœurs). Vous pouvez aussi en obtenir via la boutique.

● Un bug dans une question ?

N'hésitez pas à nous signaler toute erreur de contenu pour que nous puissions la corriger rapidement.
